## Переменные окружения

После деплоя на Vercel необходимо добавить следующие переменные окружения в настройках проекта:

1. Перейдите в **Settings** → **Environment Variables**

### Как добавить переменные:

1. В панели Vercel выберите ваш проект
2. Перейдите в **Settings** → **Environment Variables**
3. Нажмите **Add New**
4. Введите имя переменной и значение
5. Выберите окружения: **Production**, **Preview**, **Development**
6. Нажмите **Save**
7. После добавления всех переменных, перезапустите деплоймент

## После настройки переменных:

1. Перейдите в **Deployments**
2. Найдите последний деплоймент
3. Нажмите на три точки (⋯) → **Redeploy**

Или просто сделайте новый коммит и пуш в GitHub - Vercel автоматически пересоберёт проект.
